# Habbo Room in Python - WIP!

A simple room generator in python where you can create a habbo room. 

![Rotate Furni](https://raw.githubusercontent.com/wielandino/habbo-room/refs/heads/master/rotatefurni.gif =400x400)
![Habbo Room](https://raw.githubusercontent.com/wielandino/habbo-room/refs/heads/master/room.png =400x400)

## Planned features

- Place furnitures from Inventory
- Habbo Avatar
- Walking
- Interact with furnitures

# Installation & Running

Install dependencies through [pip](https://pypi.org/project/pip/)

```
pip install -r requirements.txt
```

Run the application with

```
python app.py
```
